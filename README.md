# 使用APIについて

画像生成部のAPIは[こちら](https://github.com/CyberRex0/5000choyen-api)のAPIを使用しています
APIと通信にはSSL/TLSを使用しています

# 開発環境

・MacBook (MacOS 10.15.6)

・XCode 12.4

・Swift 5.0

・iPhone 11 (iOS14.7.1)

・iPad mini3 (iOS 12.5.2)
